# mf-monitoring

## Установка проекта
```
npm install
```

### Запуск проекта в режиме разработки
```
1. Зпустить сервер localhost:3000 
   npm run server

2. Запустить проект localhost:8080
   npm run dev
```

### Сборка проекта
```
npm run build
```

### О проекте
```
Мониторинг конкурнетов.
Разработка таблицы с данными.

Стек технологий:
Vue 
Vuex
Vue-router
```
